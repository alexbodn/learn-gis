Useful hint

To resolve any trigger-related incompatibility you can simply try

to:

  • remove first any trigger: the best way you can follow is using

    "Courier New, monospace">DiscardGeometryColumn()

  • and then recreate again the triggers using

    "Courier New, monospace">AddGeometryColumn()

This will ensure that any metadata info and trigger will surely match expectations of your binary library current

version.
