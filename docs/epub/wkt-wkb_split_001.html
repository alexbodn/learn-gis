Common pitfalls

"I've declared a

"Courier New, monospace">MULTIPOINT-type Geometry

column;
now I absolutely have to insert a simple

"Courier New, monospace">POINT into this table,
but I get a constraint failed error ..."

Any MULTIxxxxx type can store a single elementary item: you simply have to use the

appropriate WKT syntax.
And anyway several useful type casting functions

exist.

 

SELECT

ST_GeometryType(ST_GeomFromText('MULTIPOINT(1.2345

2.3456)'));

"Courier New, monospace">MULTIPOINT
SELECT

ST_AsText(CastToMultiLineString(ST_GeomFromText('LINESTRING(1.2345

2.3456, 12.3456 23.4567)')));

"Courier New, monospace">MULTILINESTRING((1.2345 2.3456, 12.3456

23.4567))
SELECT

ST_AsText(CastToXYZM(ST_GeomFromText('POINT(1.2345

2.3456)')));
POINT

ZM(1.2345 2.3456 0 0)
SELECT

ST_AsText(CastToXY(ST_GeomFromText('POINT ZM(1.2345 2.3456 10

20)')));

"Courier New, monospace">POINT(1.2345 2.3456)
